# Generates unique list of ordered cost centers from a given array of cost center

## Input
```
[ [“100”, “Cost Center B”],  [“200”, “Cost Center A”], [“100”, “Cost Center B”], …]
```

## Output
```
[{
    “id”: “200”,
    “name”: “Cost Center A”
},
{
    “id”: “100”,
    “name”: “Cost Center B”
},
]
```
