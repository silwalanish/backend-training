
# Cost Center Sort module

Generate unique list of cost centers ordered alphabetically from a given array of cost centers.

## Example

```bash
Input: [ [“100”, “Cost Center B”],  [“200”, “Cost Center A”], [“100”, “Cost Center B”], …]

Output: [
    {
        “id”: “200”,
        “name”: “Cost Center A”
    },
    {
        “id”: “100”,
        “name”: “Cost Center B”
    },
    …
]
```
